# 자료구조란?

# 1. 정의

데이터 단위와 데이터 자체 사이의 물리적 또는 논리적인 관계라고 정의 할 수 있다.
쉽게 말해 자료를 효율적으로 이용할 수 있도록 컴퓨터에 저장하는 방법이다.

* 데이터 단위
  : 데이터를 구성하는 한 덩어리

# 자료구조의 분류

자료구조는 크게 선형 자료구조와 비선형 자료구조로 나뉜다.

선형 자료구조의 경우 데이터가 일렬로 나열되어 있는 것을 뜻하고,
비 선형 자료구조는 특정한 형태를 띄고 있는 것을 뜻한다.

  • 선형구조

    • 배열(Array)

    • 연결 리스트(Linked List)

    • 스택(Stack)

    • 큐(Queue)

  • 비선형 구조

    • 트리(Tree)

    • 그래프(Graph)
